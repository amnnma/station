# Command Line

| Command | Description |
| ----------- | ----------- |
| cd          | เปลี่ยน directory |
| ls | แสดงข้อมูล directory |
| pwd | แสดง directory ที่กำลังใช้งาน|
| mkdir | สร้าง directory |
| cat | แสดงข้อมูลภายในfile |
| man| แสดงคู่มือการใช้งานของcommand |
| less |  แสดงข้อมูลภายในfileโดยแสดงเป็นหน้า |
| rm | ลบfile |
| echo | เรียกinput |
